# Hoops — Web Build

Live build of the Hoops three.js gameplay test. **Play it: https://smithmw7.github.io/hoopsweb/**

Drag and flick to throw. Backtick key (or 3-finger tap on mobile) opens the debug panel
with the auto-thrower, input calibrator, and live tuning.

This repo contains **build output only** — it is redeployed from the private Hoops
repo's `three-port/` project (`npm run build`), where the source, asset pipeline, and
versioned throw configs live.
